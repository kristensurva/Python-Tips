Introduction

What is Python?

Python is the fastest growing programming language in the world right now and it is on its way to become the most popular programming language.
Python is used for data analysis, artificial intelligence and machine learning, automation.

Why use Python?

Python has a simple and beginner-friendly syntax. You can write your program with fewer lines of code and with less time.
To use Python you don't have to be a software developer, Python is also great for accountants and mathematicians for example, because with Python you can automatize lots of your daily tasks.

Indentation

In Python indentation is very important. Python uses whitespace indentation to separate code into blocks.

Variables

Let's look at an example "x = 2". This translates to variable name x receives a numeric value 2.

Statements

With statements you can write your code.
Some of the most important statements are:

  • "if" - it conditionally executes a block of code, along with "else" and "elif"(which is "else" and "if" put together)
  • "for" - iterates over an iterable object, capturing each element to a local variable for use by the attached block.
  • "while" - executes a block of code as long as its condition is true.
  • "print" - with this statement you can print whatever you want.
  • "import" - it is used for importing modules.
