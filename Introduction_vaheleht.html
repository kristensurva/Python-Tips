Introduction

Indentation

In Python indentation is very important. Python uses whitespace indentation to separate code into blocks.

Variables

Let's look at an example "x = 2". This translates to variable name x receives a numeric value 2.

Statements

With statements you can write your code.
Some of the most important statements are:

  • "if" - it conditionally executes a block of code, along with "else" and "elif"(which is "else" and "if" put together)
  • "for" - iterates over an iterable object, capturing each element to a local variable for use by the attached block.
  • "while" - executes a block of code as long as its condition is true.
  • "import" - it is used for importing modules.
