What is Python?

Python is the fastest growing programming language in the world right now and it is on its way to become the most popular programming language.
Python is used for data analysis, artificial intelligence and machine learning, automation.

Why use Python?

Python has a simple and beginner-friendly syntax. You can write your program with fewer lines of code and with less time.
To use Python you don't have to be a software developer, Python is also great for accountants and mathematicians for example, because with Python you can automatize lots of your daily tasks.
