Thonny

What's Thonny?

Thonny is an integrated development environment (IDE) for Python. It is suitable for beginners.
Thonny works on Windows, macOS and Linux.

The main author of Thonny is University of Tartu's professor Aivar Annamaa.

Features

  • Line numbers
  • Separate windows for excecuting function calls
  • Live variables during debugging
  • Simple pip GUI
